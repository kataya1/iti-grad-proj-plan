making a local branch from a remote branch (need verifying)

  1. cloning doesn't download every branch. just the main 
    git clone repo_link
  2. update the local list of remote branches 
    git remote update origin --prune
  3. check if there is a remote(should be there) 
    git remote -v
  4. if no remote exists add remote 
    git remote add origin remote_link
  5. fetch the changes of the remote branches pulling remote branch (origin is the remote repo alias) 
    git fetch origin
